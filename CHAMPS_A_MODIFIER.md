## Notes importantes

### Caractères spéciaux
Les caractères spéciaux dans le JSON sont encodés :
- `é` → `\u00e9`
- `à` → `\u00e0`
- `è` → `\u00e8`
- `'` → `\u2019` (apostrophe typographique)

### Incohérences détectées
1. **kb_avantages_style2_subtitle** : Mentionne "Paris" au lieu de "Rouen"
2. Les liens internes pointent vers `staging.site.keobiz.fr`

### Structure JSON
Tous les champs du `post_content` sont imbriqués dans des blocs WordPress Gutenberg ACF :
```
post_content → "<!-- wp:acf/bloc-name {...} -->" → data → champ
```

Les modifications devront être faites :
1. Dans le `post_content` (JSON encodé dans les blocs)
2. Avec les caractères Unicode correctement encodés
3. En préservant la structure HTML des champs
